# Like Your Car

Draft landing page for **likeyourcar.com** — an independent, personalized car-buying advisory service that helps buyers find a vehicle that fits their real needs.

> **Note:** This is a design draft. The tier names, prices, and copy are placeholder content meant to demonstrate the layout, structure, and visual direction. They will be replaced with the real service details.

## Viewing

Open `index.html` in any web browser. The page is a single self-contained file — styles and graphics are inline; only the web fonts load from the internet.

## Roadmap

- **Layer 1 — Marketing site** (this draft): polished, responsive landing page.
- **Layer 2 — Payments & survey:** Stripe checkout for the service tiers, plus a Tally survey embed that automatically emails the customer a copy of their responses.
- **Layer 3 — Freelancer portal (later):** logins and a shared queue so multiple car journalists can pick up customer surveys.

The production build will move to Astro/Next + Vercel to support the dynamic features above.
